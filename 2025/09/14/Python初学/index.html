列表、元组、字典、集合

1.核心特点

Python 中列表、元组、字典、集合是常用的内置数据结构，核心特点如下：
列表 (list) []有序、可变、可重复，支持索引、切片操作
元组 (Tuple) ()有序、不可变、可重复，比列表节省内存
字典 (Dictionary) {:}无序，键不可变、不可重复，查询效率高
集合 (Set) {} /set ()无序、可变、不可重复，支持交、并集等操作
（补充说明：字典的 “不可变 / 不可重复” 是针对键而言，值可修改；集合支持交、并、差等集合运算。）
