以下载Anaconda为例

首先下载Anaconda

1.1官网链接如下：
Anaconda官网
然后点击Get Started


在注册/登录成功以后，我们会看见两种下载：
Distribution Installers 和 Miniconda Installers，前者是全量套餐，包含 conda + Python + 数百个常用预装包 + 图形界面工具。而后者仅包含conda + Python（基础解释器） 的精简版安装程序，按自己的需求选择。

本人作为新手，还是选择Distribution Installers，避免手动下载依赖包，Python版本为3.13

1.2在清华大学开源软件镜像站下载更快：
清华大学开源软件镜像站Anaconda
2.下载完毕后打开文件，会出现一个Setup的小窗口。
阅读协议时选择I Agree
Install for:一般选择Just Me
下一个选择存储位置，若C盘空间不够可点击Browse…换为D盘然后新建文件夹。
点击next>后出现四个选项，翻译（豆包提供）如下：

  1. Create shortcuts (supported packages only)
    含义：创建快捷方式（仅支持的程序包）。
    作用：勾选后，安装完成会为 Anaconda 相关的 “支持程序”（如 Anaconda Navigator、Spyder IDE 等）在开始菜单、桌面等位置创建快捷方式，方便快速启动这些工具。
  2. Add Anaconda3 to my PATH environment variable
    含义：将 Anaconda3 添加到系统的 PATH 环境变量中。
    作用：PATH是 Windows 用来查找 “可执行程序”（如python、conda命令）的路径列表。
    提示：“NOT recommended（不推荐）” —— 因为如果系统中已有其他 Python 环境（比如单独安装的 Python），把 Anaconda 的路径加入PATH可能导致版本冲突（比如在命令行中输入python，系统不知道该调用 Anaconda 的 Python 还是其他 Python）。官方建议：用开始菜单里的「Anaconda Prompt」或「Anaconda Powershell」来操作，这些终端会自动配置好 Anaconda 的环境，避免冲突。
  3. Register Anaconda3 as my default Python 3.13
    含义：将 Anaconda3 注册为系统 “默认的 Python 3.13”。
    作用：勾选后，其他程序（如 VSCode、PyCharm 等 IDE）会自动检测到 Anaconda 的 Python，并将其作为 “系统主要的 Python 3.13 版本”。
    提示：“Recommended（推荐）” —— 适合希望用 Anaconda 作为主要 Python 环境的用户，这样在 IDE 中配置 Python 解释器时会更方便（无需手动指定路径）。
  4. Clear the package cache upon completion
    含义：安装完成后清除 “包缓存”。
    作用：“包缓存” 是安装过程中下载的临时安装包（安装完成后就没用了）。勾选后能回收磁盘空间，且不会影响 Anaconda 的功能。
    提示：“Recommended（推荐）” —— 纯优化磁盘空间的操作，建议勾选。

3.下载完成后，next>next>finish

手动配置环境变量

1.按 Win + R（Windows 键 + R 键），打开运行对话框；
2.输入 sysdm.cpl 并回车，直接打开「系统属性」的高级选项卡；
3.在「系统属性」窗口中，点击「环境变量」按钮，即可进入配置界面。
4.在用户变量中点击Path选择编辑->新建：

D:\Anaconda(你的存储路径)
D:\Anaconda\Scripts(以你的实际存储路径为准)
D:\Anaconda\Library\bin(以你的实际存储路径为准)

5.检查是否设置成功：Win + R输入cmd,然后输入conda,查看是否会蹦出相关信息

列表、元组、字典、集合

1.核心特点

Python 中列表、元组、字典、集合是常用的内置数据结构，核心特点如下：
列表 (list) []有序、可变、可重复，支持索引、切片操作
元组 (Tuple) ()有序、不可变、可重复，比列表节省内存
字典 (Dictionary) {:}无序，键不可变、不可重复，查询效率高
集合 (Set) {} /set ()无序、可变、不可重复，支持交、并集等操作
（补充说明：字典的 “不可变 / 不可重复” 是针对键而言，值可修改；集合支持交、并、差等集合运算。）
